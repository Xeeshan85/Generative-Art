Generative Fractals: An Artistic and Mathematical Journey

Fractals are infinitely complex patterns that are self-similar across different scales. They are created by repeating a simple process over and over in an ongoing feedback loop. Fractals are not just beautiful—they are deeply connected to nature, chaos theory, and complex systems.

Simple Fractals

  • Von Koch Snowflake: An equilateral triangle recursively subdivided. Equation: Replace middle third of each line segment with two sides of an equilateral triangle.
    Rule: Each segment → 4 segments (length/3)
  • Fractal Tree: Recursive binary branching, each branch splits into two.
    Angle-based recursive drawing using trigonometry.
  • Barnsley's Fern: IFS-based fern using affine transformations.
    xₙ₊₁ = a*xₙ + b*yₙ + e, yₙ₊₁ = c*xₙ + d*yₙ + f
  • Dragon Curve: Recursive folding curve. Turns alternate left and right.
    Drawn via L-systems: F → F+G, G → F-G
  • Sierpinski Triangle: Triangle subdivided into smaller triangles.
    Remove central triangle in each iteration.
  • Sierpinski Carpet: Square divided into 9, center removed recursively.
    Fractal dimension: log(8)/log(3)

Complex Fractals

  • Mandelbrot Set z¹, z², z³, z⁴: Iteration: zₖ → zₖ^n + c. If |z| escapes to infinity, point is not in the set. Higher powers increase complexity.
  • Burning Ship: Iteration: z = (|Re(z)| + i|Im(z)|)^2 + c. Creates flame-like structure.
  • Julia Set: Similar to Mandelbrot, but with fixed c. zₖ+1 = zₖ^2 + c. Varies wildly with c.
  • Phoenix: Modified Mandelbrot: zₖ+1 = zₖ^2 + c + p*zₖ⁻¹
  • Tricorn: Iteration: zₖ+1 = ̅zₖ^2 + c (conjugate Mandelbrot)
  • Multibrot: Generalized Mandelbrot: zₖ+1 = zₖ^d + c (d > 2)
  • Newton Fractal: Root-finding fractals: zₖ+1 = z - f(z)/f'(z). Behavior depends on roots of f(z).
  • Rose Curve: Polar equation: r = cos(kθ) or r = sin(kθ). Forms petal shapes.
  • Hypocycloid: Curve traced by a point on a circle rolling inside another circle.
    Parametric: x = (a - b)cos(t) + b cos((a - b)t/b)
  • Fermat Spiral: Spiral where radius grows with sqrt of angle.
    r = √θ or x = √θ cos(θ), y = √θ sin(θ)

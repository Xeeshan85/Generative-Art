Welcome to Generative Art

This project showcases various fractal visualizations using HTML and JavaScript.

  • Fractal Renderer (Simple)
    Includes classic fractals like:
    • Von Koch Snowflake
    • Dragon Curve
    • Fractal Tree
    • Barnsley Fern
    • Other basic recursive patterns
  • Fractal Renderer (Complex)
    Includes complex fractals based on iterative functions:
    • Mandelbrot Set
    • Julia Set
    • Newton Fractals
    • Burning Ship
    • Other escape-time fractals

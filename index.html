Generative Fractals: An Artistic and Mathematical Journey

Fractals are infinitely complex patterns that repeat across different scales. They are generated by iterating simple rules, resulting in intricate shapes found in nature and chaos theory.

Simple Fractals

  • Von Koch Snowflake: Start with a triangle; recursively divide each line into thirds and build a peak in the middle.
    Rule: Each segment â†’ 4 segments of length \( \frac{1}{3} \)
  • Fractal Tree: Binary tree where each branch splits into two smaller ones using trigonometry.
  • Barnsley's Fern: A fern made using affine transformations:
    Equations: \( x_{n+1} = ax_n + by_n + e \), \( y_{n+1} = cx_n + dy_n + f \)
  • Dragon Curve: Recursive folding pattern:
    L-system: \( F \rightarrow F+G,\quad G \rightarrow F-G \)
  • Sierpinski Triangle: Repeatedly remove the center triangle from a larger one.
  • Sierpinski Carpet: Divide square into 9 parts, remove the center recursively.
    Fractal dimension: \( \frac{\log(8)}{\log(3)} \)

Complex Fractals

  • Mandelbrot Set (e.g., \( z^2, z^3, z^4 \)): Iterate: \( z_{k+1} = z_k^n + c \).
    If \( |z| \to \infty \), point is not in the set.
  • Burning Ship: Iterate: \( z = (|\text{Re}(z)| + i|\text{Im}(z)|)^2 + c \)
  • Julia Set: Like Mandelbrot, but uses a fixed \( c \): \( z_{k+1} = z_k^2 + c \)
  • Phoenix: Modified Mandelbrot: \( z_{k+1} = z_k^2 + c + p z_{k-1} \)
  • Tricorn: Uses complex conjugate: \( z_{k+1} = \overline{z_k}^2 + c \)
  • Multibrot: Generalized form: \( z_{k+1} = z_k^d + c \), where \( d > 2 \)
  • Newton Fractal: Based on Newton's method:
    \( z_{k+1} = z_k - \frac{f(z_k)}{f'(z_k)} \)
  • Rose Curve: Polar plot of:
    \( r = \cos(k\theta) \) or \( r = \sin(k\theta) \)
  • Hypocycloid: Curve from a point on a circle rolling inside another:
    \( x = (a - b)\cos(t) + b\cos\left(\frac{a - b}{b}t\right) \)
  • Fermat Spiral: Spiral where radius grows with square root of angle:
    \( r = \sqrt{\theta} \), or \( x = \sqrt{\theta}\cos(\theta),\ y = \sqrt{\theta}\sin(\theta) \)
